What is Discord?

Discord is a free VoIP platform targeted at gamers and gaming communities. Specifically they offer voice and text chat channels on servers, and users can use either an installable client or a web based interface.

PandaBot

PandaBot is a bot utilizing JDA, a Java implementation of Discord's API. PandaBot primarily exists as a music player, at its core utilizing a stripped down and slightly modified version of the music player from Frederikam's Discord bot, Fredboat. PandaBot also carries additional functions such as a NSFW command for masking NSFW material or spoilers.

PandaBot is currently a standalone bot that is not publicly hosted, and consequently has a setup process involved if you want to use it. Fortunately, said setup is surface level and does not require extensive technological knowledge. All necessary information is available in the bot's ReadMe on GitHub.
