What is Bukkit, and what are Bukkit Plugins?

Bukkit is a project that modifies the Minecraft server. While it only makes a few tweaks on its own, it supports plugins that could do much more with no client modding required. Since Bukkit's life was unexpectedly cut short by some political nonsense that is longer than I feel like writing, the Spigot project has picked up the pieces and kept Bukkit's legacy alive. I still refer to them as Bukkit plugins, just because that's what they were all along.

Plugin Releases

HorseStats

My most successful plugin, HorseStats, was originally built to provide an interface for players to see technical information about horses. It has since been expanded on, with features for protection, teleportation and more.

SimpleEgg

This plugin was born as a response to when the MobCatcher plugin was moved to the Spigot resources page, and the full version was no longer a free utility. It does almost all the things MobCatcher did, with a few minor differences on occasion. It is NMS free, meaning it doesn't require an exact server version to run.

FernMode

FernMode is a lighter, silly plugin that lets players turn into ferns with a command. Nothing more to it.

PlateSync

PlateSync is a lightweight plugin that makes adjacent pressure plates activate each other when one is pressed. No configuration, no commands, no permissions. It just works.
